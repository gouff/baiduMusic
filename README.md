##介绍
本项目可以解决这一问题。

当然有三个前提：

- 该音乐有无损版本
- 已经登陆
- 加入了红心收藏


**已经部署在新浪 SAE **: http://bbmusic.sinaapp.com/  



主要代码在index.html的JS部分，很简单。如果你希望用户**免登陆，且不需要直接加入收藏，直接提供地址即可**，只需要一个**已知账号收收藏了足够多的音乐即可**。
